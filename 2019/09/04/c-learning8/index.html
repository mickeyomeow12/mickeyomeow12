数组

定义数组

格式： <类型> 变量名称[元素数量]

先有数组每个单元的类型 int char 等

方括号表明是个数组 里面数字是数量

特点

  • 其中所有的元素具有相同的数据类型；

  • ⼀旦创建，不能改变⼤⼩

  • 顺序储存

  • 数组从0开始

  • 赋值号左边叫左值

  • 数组的每个单元就是数组类型的⼀个变量

  • 使⽤数组时放在[]中的数字叫做下标或索引，下标从0开始计数

  • 最大的下标是数组个数减一 习惯从0开始数

数组的集成初始化

直接⽤⼤括号给出数组的所有元素的初始值

如果给出了数组的⼤⼩，但是后⾯的初始值数量不⾜，则其后的元素被初始化为0

集成初始化的定位

  • ⽤[n]在初始化数据中给出定位
  • 没有定位的数据接在前⾯的位置后⾯
  • 其他位置的值补零
  • 也可以不给出数组⼤⼩，让编译器算
    算法是按照初始化的下标给的多少算
  • 特别适合初始数据稀疏的数组

数组的大小 （在集成初始化的前提下）

sizeof给出整个数组所占据的内容的⼤⼩，单位是字节

方法：sizeof(a)/sizeof(a[0])

sizeof(a[0])给出数组中单个元素的⼤⼩，于是相除就得到了数组的单元个数

集成初始化最后一个数字加逗号

数组的赋值

数组不能直接赋值

int a[ ]={2,3,4}

int b[ ]=a;不可以这样⬅️

数组变量本身不能被赋值

要把⼀个数组的所有元素交给另⼀个数组，必须采⽤遍历

一个数组给另个数组赋值的方法：

遍历数组的时候都用for循环，让循环变量i从0到<数组的⻓
度，这样循环体内最⼤的i正好是数组最⼤的有效下标

注意

  • 循环结束条件不可以是<=数组⻓度，
  • 离开循环后，不可以继续⽤i的值来做数组元素的下标！（因为离开的时候那个下标已经变成无效下标了）

数组作为函数的参数时：（自己写的参数的时候）

  • 不能在[]中给出数组的⼤⼩ （就是void /int 函数名（参数）里的参数这一块不能再[]里填数字）
  • 不能再利⽤sizeof来计算数组的元素个数！

构造素数表

欲构造n以内的素数表

  1. 令x为2
  2. 将2x、3x、4x直⾄ax<n的数标记为⾮素数
  3. 令x为下⼀个没有被标记为⾮素数的数，重复2；直
    到所有的数都已经尝试完毕

伪代码实现：

  1. 开辟prime[n]，初始化其所有元素为1，prime[x]为1
    表⽰x是素数
  2. 令x=2
  3. 如果x是素数，则对于(i=2;xi<n;i++)令prime[ix]=0
  4. 令x++，如果x<n，重复3，否则结束
