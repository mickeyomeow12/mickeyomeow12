数据类型

C是有类型的语⾔

C语⾔的变量，必须：

  • 在使⽤前定义，并且
  • 确定类型

类型不同

所表达的数的范围：char < short < int < float < double

sizeof

  • 是⼀个运算符，给出某个类型或变量在内存中所占据的字节数
  • sizeof(int)
  • sizeof(i)

sizeof 可以了解类型在内存中占据多大的空间

sizeof 是静态运算符，它的结果在编译时刻就决定了

注意不要在sizeof的括号⾥做运算，这个函数并不会做运算

浮点类型

在%和f之间加上.n可以指定输出⼩数点后⼏位，这样
的输出是做4舍5⼊的
• printf(“%.3f\n”, -0.0049);
• printf(“%.30f\n”, -0.0049);
• printf(“%.3f\n”, -0.00049);

浮点运算的精度

  • 带⼩数点的字⾯量是double⽽⾮float
  • float需要⽤f或F后缀来表明⾝份
  • 如果没有特殊需要，只使⽤double

字符类型

char是⼀种整数，也是⼀种特殊的类型：字符。

  • ⽤单引号表⽰的字符字⾯量：’a’, ‘1’
  • ‘’也是⼀个字符
  • printf和scanf⾥⽤%c来输⼊输出字符

混合输入

scanf(“%d %c”, &i, &c);
读完整数要把空格都读掉

scanf(“%d%c”, &i, &c);
读完整数就不读了

字符计算

⼀个字符加⼀个数字得到ASCII码表中那个数之后的字符

两个字符的减，得到它们在表中的距离

大小写转换

字⺟在ASCII表中是顺序排列的

  • ⼤写字⺟和⼩写字⺟是分开排列的，并不在⼀起
  • ‘a’-‘A’可以得到两段之间的距离，于是
  • a+’a’-‘A”可以把⼀个⼤写字⺟变成⼩写字⺟，⽽
  • a+’A’-‘a’可以把⼀个⼩写字⺟变成⼤写字⺟

自动类型转换

对于printf，任何⼩于int的类型会被转换成int；float会被转换成double

但是scanf不会，要输⼊short，需要%hd

强制类型转换

要把⼀个量强制转换成另⼀个类型（通常是较⼩的类型），需要：

  • (类型)值
  • ⽐如：
    • (int)10.2
    • (short)32
  • 注意这时候的安全性，⼩的变量不总能表达⼤的量
    • (short)32768

double a = 1.0;
double b = 2.0;
int i = (int)a / b; 只把a强制转换成int

int i = (int)(a / b); 把a和b都强制类型转换

强制类型转换的优先级⾼于四则运算

逻辑运算

逻辑运算是对逻辑量进⾏的运算，结果只有0或1

优先级

! > && > ||

!done && (count > MAX)

短路

逻辑运算是⾃左向右进⾏的，如果左边的结果已经能
够决定结果了，就不会做右边的计算
• a==6 && b==1
• a==6 && b+=1

对于&&，左边是false时就不做右边了

对于||，左边是true时就不做右边了

不要把赋值，包括复合赋值组合进表达式

条件运算符

不要用嵌套的条件表达式

count = (count > 20) ? count -10 : count+10;

条件、条件满⾜时的值和条件不满⾜时的值
if ( count >20 )
count = count-10;
else
count = count+10;

条件运算符的优先级⾼于赋值运算符，但是低于其他运算符
