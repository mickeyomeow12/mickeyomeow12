For 语句：

For(初始条件;循环继续的循环条件;循环每轮要做的操作){

}

For循环像⼀个计数循环：

  • 设定⼀个计数器，初始化它，（第一个表达式）
  • 然后在计数器到达某值之前，（第二个表达式）重复执⾏循环体，
  • ⽽每执⾏⼀轮循环，计数器值以⼀定步进（第三个表达式）进⾏调整，
    ⽐如加1或者减1

For = 对于

for ( count=10; count>0; count– )

就读成：“对于⼀开始的count=10（第一个表达式 处事），当
count>0时，重复做循环体（第二个表达式 判断），每⼀轮循环
在做完循环体内语句后，使得count–。（第三个表达式）
”

注意for循环内的条件判断（第二个表达式）是在一开始循环时做的（同while）

For小套路

  • 做求和的程序时，记录结果的变量应该初始化为0
  • 做求积的变量时，记录结果的变量应该初始化为1（如果求积也为0那么怎么乘都是0）

循环次数

常用 ： for ( i=0; i< n; i++ )

循环次数是多少？
对于这个表达式 循环的次数是n，⽽循环结束以后，i的
值是n。（n是循环内不不能达到的值，循环结束n的值一定是n）

(i=1;i<=n;i++)和上方表达式同理 循环次数都是n 不同的是第一个i是多少和离开时的i

注意循环的控制变量i，是选择从0开始 还是从1开始，是判断i < n还是判断i<=n，对循环的次数，循环结束后变量的值都有
影响

Tips for loops

  • 如果有固定次数，⽤for
  • 如果必须执⾏⼀次，⽤do_while
  • 其他情况⽤while
