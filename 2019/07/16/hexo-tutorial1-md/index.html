入门

常用Hexo命令

初始化目录：hexo init [folder]

新建文章：hexo new [layout] 或 hexo n [layout] <title>

layout 指的是格式

生成网页

生成静态文件：hexo generate 或 hexo g

监听文件变化：hexo g –watch 或 hexo g -w

远程上传同步git

部署：hexo deploy 或 hexo d

先生成后部署：hexo d -g

部署查看本地网页动态

启动本地服务器（服务器会监听文件变化并自动更新）：hexo server 或 hexo s

启动调试：hexo s –debug

预览草稿：hexo s –draft

清除缓存：hexo clean

网页

hexo new page_name

日志

新建一个日志

hexo new article_name

新建一个草稿

hexo new draft article_name

run with draft : hexo server –draft
