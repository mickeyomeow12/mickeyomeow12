入门

常用Hexo命令

初始化目录：hexo init [folder]

新建文章：hexo new [layout] 或 hexo n [layout] <title>

layout 指的是格式

生成网页

生成静态文件：hexo generate 或 hexo g

监听文件变化：hexo g –watch 或 hexo g -w

远程上传同步git

部署：hexo deploy 或 hexo d

先生成后部署：hexo d -g

部署查看本地网页动态

启动本地服务器（服务器会监听文件变化并自动更新）：hexo server 或 hexo s

启动调试：hexo s –debug

预览草稿：hexo s –draft

清除缓存：hexo clean

网页

hexo new page_name

日志

新建一个日志

hexo new article_name

新建一个草稿

hexo new draft article_name

run with draft : hexo server –draft

官方指路

配置

第一步：在终端找到你的hexo博客的文件夹的目录 然后输入下方命令

$ npm install hexo-helper-live2d –save

在 插件作者的博客里挑选一只你喜欢的看板娘，记下它的名字

第二步：live2d-widget-model-你选中的萌物名字 即可获得对应的 Live2D 资源

以 tororo 这只白喵为例

仍是在终端的hexo博客根目录输入命令

$ npm install live2d-widget-model-tororo –save

第三步：配置博客的_config.yml文档

如何加密自己的hexo博文

第一步 安装：

npm install –save hexo-blog-encrypt

第二步 快速开始：

  • 首先, 你需要确保你的文章中含有内容（不能为空，或者只包含空格）

  • 在 _config.yml 中启用该插件

第三步 给文章添加密码

  • password: 是该博客加密使用的密码
  • abstract: 是该博客的摘要，会显示在博客的列表页
  • message: 这个是博客查看时，密码输入框上面的描述性文字
