重头学c语言-1

程序框架

所有程序都需要这个框架

变量

变量是⼀个保存数据的地⽅，当我们需要在程序⾥保存数据时，就需要一个变量来保存那个数据

变量定义的⼀般形式： <类型名称> <变量名称>; 最后不要忘了分号

变量需要⼀个名字，变量的名字——“标识符”，意思是它是⽤来识别这个和那个的不同的名字。

小写

标识符有标识符的构造规则。

基本的原则是：标识符只能由字⺟、数字和下划线组成，数字不可以出现在第⼀个位置上，C语⾔的关键字不可以⽤做标识符。

如： int price = 0;

这⼀⾏，定义了⼀个变量。变量的名字是price，类型是int，初始值是0。

等号是赋值，把右边的东西给左边

初始化

当赋值发⽣在定义变量的时候，就像int price = 0，就是变量的初始化。
虽然C并没有强制要求所有的变量都在定义的地⽅做初始化，
但是所有的变量在 第⼀次被使⽤（出现在赋值运算符的右边)之前 应该被赋值⼀次

没有初始化就会被乱赋值

变量初始化

<类型名称> <变量名称> = <初始值>;

int price = 0;

int amount = 100;

组合变量(一行定义多个变量）

注意：必须一个一个赋值 不能price,amount=0这样

定义的时候，也可以在这个定义中单独给单个变量赋初值，如：

• int price = 0, amount = 100;

常量

是固定不变的数，是常数。在程序⾥，我们称作直接量（literal）。

全大写

int change = 100 - price;
如这行中的100即为常量，直接量

更好的⽅式，是定义⼀个常量：

const int AMOUNT = 100;

const是⼀个修饰符，加在int的前⾯，⽤来给这个变量加上⼀个const（不变的）的属性。

这个const的属性表⽰这个变量的值⼀旦初始化，就不能再修改了。

int change = AMOUNT - price;

scanf函数

scanf()

读数%d 小心变量前需要加&

程序要求读⼊多个数字时，可以在⼀⾏输⼊，中间⽤空格分开，也可以在多⾏输⼊

printf（） 输出函数

说明后头有个值需要输出在这个位置 把后头的值填进去

双引号里头原封不动输出

四则运算 注意 取余(两个数相除以后的余数)用百分号% 除用/

浮点数：

因为两个整数的运算的结果只能是整数

所以要用小数点运算的时候要改成x.x 或者设定为double/float

10和10.0在C中是完全不同的数

带小数点和不带小数点的数是完全不同的两个数

10.0是浮点数

double

float

表达式

⼀个表达式是⼀系列运算符和算⼦的组合，⽤来计算⼀个值

count = count + 1；

整个这一行都是表达式

运算符

运算符（operator）是指进⾏运算的动作，⽐如加法运算符“+”，减法运算符“-”，“=”赋值。

算⼦（operand）是指参与运算的值，这个值可能是常数，也可能是变量，还可能是⼀个⽅法的返回值

单目的运算 + - 是当作正负号看待的 一级优先运算符

赋值运算符：

自右向左的运算

赋值也是运算，也有结果

a=6的结果是a被赋予的值，也就是6

a=b=6 —> a=(b=6)

不要做嵌入式赋值！！

一些杂七杂八的运算问题

因为%百分号已经被当做取余的百分号

所以题目若是给关于百分之多少的运算时不能直接用xx% 而是要转换成小数再运算

c语言没有做幂次的运算符 次方只能乘几遍

一些代码

计算时间差

int hour1,mins1;
int hour2,mins2;

printf("pls enter the times \n");
scanf("%d %d",&hour1,&mins1);
scanf("%d %d",&hour2,&mins2);

int t1 = hour1*60+mins1;//c99可以直接在int的时候直接做运算
int t2 = hour2*60+mins2;//c99可以直接在int的时候直接做运算
//hour1*60+minute1—>转换为分钟为单位

int t3 = t2-t1; //c99可以直接在int的时候直接做运算
//t1,t2与t3不同部分所以应该回车分成不同部分

printf("时间差为%d小时%d分钟\n",t3/60,t3%60 );
//t/60—>⼩时部分；t%60—>分钟

复合赋值

5个算术运算符，+ - * / %，可以和赋值运算符“=”结合起来，
形成复合赋值运算符：“+=”、“-=”、“*=”、“/=”和“%=”

total += 5;
total = total + 5;

注意 两个运算符中间不要有空格

递增递减运算符

“++”和“–”是两个很特殊的运算符，它们是单目运算符，这个算⼦还必须是变量。

这两个运算符分别叫做递增和递减运算符，他们的作⽤就是给这个变量+1或者-1

count++;
     等于 count += 1;
     等于 count = count + 1;

前缀后缀

可以单独使用 不可以组合使用

++和- -可以放在变量的前⾯，叫做前缀形式

a++

a++作为一个表达式表达的值是a加1以前的值

下一步a变成a+1的值了

++和- -也可以放在变量的后⾯，叫做后缀形式。

++a

++a的表达式的值是加了1以后的值

下一步a不变了


表达式 运算 表达式的值
count++ 给count加1 count原来的值
++count 给count加1 count+1以后的值
count- - 给count减1 count原来的值
- -count 给count减1 count-1以后的值
