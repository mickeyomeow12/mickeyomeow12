判断

如果 if
⼀个基本的if语句由⼀个关键字if开头，跟上在括号⾥的⼀个表⽰条件的逻辑表达式，然后是⼀对⼤括号“{}”之间的若干条语句。

如果表⽰条件的逻辑表达式的结果不是零，那么就执⾏后⾯跟着的这对⼤括号中的语句，
否则就跳过这些语句不执⾏，⽽继续下⾯的其他语句.

建议：if 和 else后总是用{}

格式：

if 语句的圆括号后面要么有大括号 要么是一句一句 圆括号的右括号后面不能有分号

if ( total > amount )
total += amount+10;

注意

if和else后面加大括号的重要性：

  1. 可以使得代码逻辑更加清晰
  2. 如果不加大括号则if和else后面只能运行后面的一条语句加上之后可以将大括号内的内容看成是一条语句，运行大括号内的所有内容。

当if else没有大括号时：

if语句这⼀⾏结束的时候并没有表⽰语句结束的“;”，
⽽后⾯的赋值语句写在if的下⼀⾏，并且缩进了，
在这⼀⾏结束的时候有⼀个表⽰语句结束的“;”。

这表明这条赋值语句是if语句的⼀部分，if语句拥有和控制这条赋值语句，
决定它是否要被执⾏。

}

关系运算

计算两个值之间的关系

运算符 意义
＝＝ 相等
!= 不相等
> ⼤于
>= ⼤于或等于
< ⼩于
<= ⼩于或等于

因为是两个值的计算 所以一定有结果

当两个值的关系符合关系运算符时（即运算成立），关系运算的结果为整数1，否则为整数0

优先级

运算的时候要注意！ 这是重点！

所有的关系运算符的优先级⽐算术运算的低，但是⽐赋值运算的⾼

7 >= 3 + 4
//先算3+4=7 算术运算符先算 再算关系运算符 7>=7 所以答案为1

int r = a>0;
//先判断a>0过去 再算赋值运算符r=1/0

注意：
判断是否相等的==和!＝的优先级⽐其他的 低，⽽连续的关系运算是从左到右进⾏的

Switch-Case

  • 控制表达式只能是整数型的结果
  • 常量可以是常数，也可以是常数计算的表达式
  • 根据表达式的结果，寻找匹配的case，
    并执⾏case后⾯的语句，⼀直到break为⽌
  • 如果所有的case都不匹配，那么就执⾏default后⾯的语句；
    如果没有default，那么就什么都不做

switch语句可以看作是⼀种基于计算的跳转，计算控制表达式的值后，程序会跳转到相匹配的case（分⽀标号）处。
分⽀标号只是说明switch内部位置的路标，在执⾏完分⽀中的最后⼀条语句后，如果后⾯没有break，就会顺序执⾏到下⾯的case⾥去，
直到遇到⼀个break，或者switch结束为⽌。

case执行完了没有遇到break那么久继续往下走 并且继续执行 直到遇到break；
